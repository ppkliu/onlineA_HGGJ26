# 國王（King）

| 項目 | 說明 |
|---|---|
| 年齡 | 48 歲 |
| 角色定位 | 缺席的父親。他的每一個決定都是為了保護城市和女兒，但代價是讓所有人誤解他 |
| 核心衝突 | 「朕不知道叛徒是誰，但朕知道叛徒存在」——所以他選擇犧牲名譽秘密備戰，而非打草驚蛇 |

## 表情組

`default`（威嚴）

## 關鍵設計

國王不直接出現在公主面前（直到最終輪迴的朝堂對質），但他的影響無處不在：徵兵令是他下的，稅是他加的，密令是他給的，地下防線是他建的。他是整個故事中最大的「善意加害者」。

他給賽拉斯的密令——在城破時殺死公主——不是冷酷，而是不願讓女兒落入敵手。他知道這道命令有多殘忍，他也知道賽拉斯會多麼痛苦。但他別無選擇。

## 玩家感知弧線

| 輪迴 | 玩家對國王的印象 |
|---|---|
| Loop 0–1 | 缺席、冷漠、壓迫人民的暴君（宰相的形象塑造） |
| Loop 2 | 間接加害者——徵兵、加稅、鎖糧倉 |
| Loop 3 | 開始懷疑，但仍無法確定他是善意還是惡意 |
| 最終輪迴 | 真相揭曉：他知道叛徒存在，選擇用犧牲自己名聲換取城市存活的機會 |
