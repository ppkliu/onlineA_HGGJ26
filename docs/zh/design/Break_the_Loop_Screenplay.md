## 閱讀指南

本文件是所有 Dialogic `.dtl` 時間線的**劇本藍圖**。格式說明：

| 標記 | 用途 |
|---|---|
| `【場景】` | 場景描述與視覺指示 |
| `【BGM】` | 背景音樂切換 |
| `【SFX】` | 音效觸發 |
| `【靜默】` | Cut to Silence |
| `【轉場】` | 畫面轉場效果 |
| `【選擇肢】` | 玩家選項（含條件鎖定標記） |
| `【情報獲得】` | 觸發 `IntelSystem.acquire_intel()` |
| `【系統】` | 系統提示訊息（如倒數天數） |
| `【內心】` | 公主的內心獨白（不出聲） |
| `（括號）` | 角色的動作、表情、語氣指示 |

角色台詞格式：`角色名（表情/動作）：「台詞內容。」`

# 第一部分：角色總表與勢力關係

## 1.1 核心角色

### 公主 艾蓮娜（Princess Elena）

| 項目 | 說明 |
|---|---|
| 年齡 | 17 歲 |
| 勢力歸屬 | 王室（但實質上不屬於任何勢力，是所有勢力的棋子） |
| 角色定位 | 玩家角色。從無知的籠中鳥，到看透一切的破局者 |
| 口癖/語氣 | 初期：天真、疑問句多。後期：冷靜、陳述句，偶爾帶著輪迴疲憊的蒼涼 |
| 表情組 | `default`（微笑）、`sad`（哀傷）、`shocked`（驚愕）、`angry`（憤怒）、`determined`（堅定）、`hollow`（空洞，多次輪迴後） |
| 核心衝突 | 她是這座城堡裡唯一「什麼都不知道」的人——而這份無知，是所有人刻意維護的 |

**角色弧線：**

- **序章**：等待被拯救的公主 → 死於自己的無知
- **第 1 輪迴**：驚恐、否認 → 試圖用「公主的身份」改變命運 → 失敗
- **第 2 輪迴**：開始放下身段 → 嘗試理解他人的痛苦 → 觸碰到真相的邊緣
- **中期輪迴**：從「為什麼沒人救我」變成「為什麼沒人告訴我」→ 理解每個人的苦衷
- **最終輪迴**：不再是等人來救的公主，而是主動拯救所有人的人

### 騎士長 賽拉斯（Knight Commander Silas）

| 項目 | 說明 |
|---|---|
| 年齡 | 34 歲 |
| 勢力歸屬 | 友善貴族（王室忠臣） |
| 角色定位 | 序章的「加害者」，真相中的「最大受害者」。整個遊戲最令人心碎的角色 |
| 口癖/語氣 | 寡言、用軍事化的簡短句。情緒崩潰時才會說長句 |
| 表情組 | `default`（嚴肅）、`tearful`（含淚）、`exhausted`（疲憊）、`relieved`（釋然） |
| 核心衝突 | 他從小看著艾蓮娜長大。國王給他的最後密令是：如果城破，親手殺死公主，不讓她落入敵手。他一直在祈禱自己不必執行這道命令 |

**關鍵設計：** 賽拉斯在日常中的每一句冷淡回應，都是因為他不敢和公主太親近——因為他知道自己可能要親手殺她。玩家在後期輪迴理解這一點後，重聽他前期的台詞會極度心痛。

### 宰相 莫里昂（Chancellor Morion）

| 項目 | 說明 |
|---|---|
| 年齡 | 52 歲 |
| 勢力歸屬 | 敵意貴族（叛國者） |
| 角色定位 | 真正的幕後黑手。玩家前期的「假盟友」 |
| 口癖/語氣 | 永遠溫和有禮，用「殿下」開頭。越危險的話說得越輕柔 |
| 表情組 | `default`（慈祥微笑）、`concerned`（假裝擔憂）、`cold`（露出真面目，僅最終輪迴） |
| 核心衝突 | 國王推行的改革削減了貴族特權。莫里昂與敵國達成密約：裡應外合攻破王城，事後保留他的封地與財富 |

**關鍵設計：** 莫里昂的台詞永遠不說謊——他只是省略真相。例如「陛下已經派人處理了」（真相：國王確實在備戰，但莫里昂在暗中破壞）。玩家在獲得情報後重讀他的話，會發現每一句都是精心設計的文字陷阱。

**關鍵設計：** 莉娜不是內鬼，她是體制的受害者。她用笑容包裝痛苦，和公主用無知包裝恐懼，是同一種悲劇的兩面。

**關鍵設計：** 達爾的眼睛就是被莫里昂弄瞎的——這個事實本身就是宰相叛國的物證。但達爾不會主動說出來，因為他怕連累收留他的窮人們。公主必須帶著足夠的情報，讓他相信「這個公主不一樣了」。

| 項目 | 說明 |
|---|---|
| 年齡 | 28 歲 |
| 勢力歸屬 | 受鼓動暴民 |
| 角色定位 | 城破之夜打開城門的內應。但他不是壞人——他是被絕望和謊言逼到絕路的普通人 |
| 口癖/語氣 | 粗獷、直接、充滿怒氣。罵的最多的詞是「那些貴族」 |

**關鍵設計：** 布魯諾是整個遊戲中最難應對的角色。他的憤怒是真實的、合理的。公主不能用「國王是好人」這種空話說服他——她必須帶來國王確實在為平民準備後路的具體證據。

# 第二部分：序章（第 0 輪迴）— 無知的毀滅

> 對應 Dialogic 時間線：`prologue/prologue_main.dtl` → `prologue_assassination.dtl` → `prologue_death.dtl`

## 2.1 場景：亡國之夜

```
時間線檔案：prologue/prologue_main.dtl
```

【場景】公主的寢房。華麗的房間被窗外沖天的火光映成血紅色。牆上的掛毯在震動中歪斜，精緻的瓷器從架上滾落摔碎。

【BGM】`prologue_epic.ogg` — 低沉的弦樂，帶有不祥的鼓點，像是心跳逐漸加速。

【SFX】遠處的爆炸聲、金屬碰撞聲、模糊的慘叫聲。所有聲音被厚重的石牆削弱，像是從另一個世界傳來。

---

艾蓮娜（shocked）：「怎、怎麼回事……外面好吵……」

【內心】我從床上坐起來。窗外的天空是不對的——不是夜晚該有的黑色，而是一種翻攪著的橘紅，像是有人把整座城點燃了。

【SFX】一聲巨響。門外走廊傳來重物倒地的聲音，然後是拖行的聲響。

艾蓮娜（shocked）：「來人！莉娜？有人在嗎？」

【內心】我跑到門邊，但門被從外面鎖住了。我拍打著門板，指節很快就開始發疼。沒有人回應。

【SFX】拍門聲。木門沉悶的回響。

艾蓮娜（sad）：「父王……父王一定會派人來的。對，父王說過，這座城堡有神明賜予的魔法守護，絕對不會有事的……」

【內心】我把自己縮在床角，抱著膝蓋。禮儀導師說過，公主不應該哭。但我的牙齒一直在打顫。

【SFX】遠處的慘叫聲突然變大——有人在走廊上奔跑。

艾蓮娜（shocked）：「是救兵嗎……？！」

## 2.2 騎士長的到來

```
時間線檔案：prologue/prologue_assassination.dtl
```

【SFX】沉重的腳步聲急促靠近。金屬盔甲的碰撞聲。然後是門鎖被暴力扭斷的尖銳聲響。

【場景】門被猛力撞開。騎士長賽拉斯站在門口，銀色的盔甲沾滿了黑色的血污，半邊披風已經燒焦。他的劍上還在滴血。背後的走廊彌漫著濃煙。

賽拉斯（tearful）：「……殿下。」

艾蓮娜（default）：「賽拉斯！太好了，你來了！父王派你來接我的對不對？快帶我離開這裡——」

【內心】我看到他了。那個從小保護我的騎士長，比任何城牆都可靠的人。我朝他跑過去。

賽拉斯（tearful）：「……」

【場景】賽拉斯沒有張開雙臂迎接她。他一步一步走進房間，反手關上了門。門關上的瞬間，走廊的嘈雜聲被隔絕了大半。

【SFX】門關上。沉悶的一聲。房間突然變得相對安靜，只剩下窗外遠處的火焰噼啪聲。

艾蓮娜（shocked）：「賽……賽拉斯？你受傷了嗎？你身上好多血……」

賽拉斯（tearful）：「殿下，城破了。」

艾蓮娜（shocked）：「城破……？怎麼會？魔法陣不是——」

賽拉斯（exhausted）：「魔法陣在三天前就失效了。有人從內部切斷了能源核心。」

艾蓮娜（shocked）：「失效了？三天前？為什麼沒有人告訴我！」

賽拉斯（tearful）：「……」

艾蓮娜（angry）：「父王呢？父王在哪裡？帶我去找父王！他一定有辦法的——」

賽拉斯（tearful）：「陛下他……」

【場景】賽拉斯單膝跪下。他的動作僵硬，像是一座正在崩塌的鐵塔。這是艾蓮娜第一次看到他跪下。

賽拉斯（tearful）：「陛下已經……戰死在城門前了。是臣……親手收的遺體。」

【SFX】一秒的完全靜默。連窗外的聲音都像是被抽走了。

艾蓮娜（shocked）：「……你說什麼？」

賽拉斯（tearful）：「外面全是敵軍和叛軍。南門、西門都被打開了——是城裡的人打開的。」

艾蓮娜（sad）：「不……不對，你騙我的對不對？父王那麼強……他怎麼會……」

賽拉斯（tearful）：「殿下，請聽臣說。」

【場景】賽拉斯緩緩站起來。他拔出了劍。劍身上的血跡在火光中發出暗紅色的光。

艾蓮娜（shocked）：「你……你在做什麼？」

賽拉斯（tearful）：「陛下最後的密令——」

【場景】他的聲音在顫抖，握劍的手也在顫抖。但劍尖指向了艾蓮娜。

賽拉斯（tearful）：「『絕不能讓公主落入敵人手中。不惜一切代價。』」

艾蓮娜（shocked）：「什麼……？你要殺我？」

艾蓮娜（angry）：「你是來救我的，不是嗎！帶我逃出去啊！外面那麼大，一定有——」

賽拉斯（tearful）：「沒有地方了，殿下。城外是十萬敵軍，城內是暴民和叛軍。所有的逃生通道都已經被切斷了。」

賽拉斯（tearful）：「臣殺了一路的敵兵才到這裡。門外的屍體……臣最多再撐半炷香。然後他們就會破門而入。」

賽拉斯（tearful）：「他們會對您做什麼……臣不敢想。陛下也不敢想。所以他給了臣這道命令。」

艾蓮娜（sad）：「不……我不要死……我不想死……」

【內心】我在後退。後背撞到了牆壁，冰冷的石頭穿透單薄的睡衣。我已經無路可退了。

艾蓮娜（angry）：「父王……父王為什麼不帶我一起走！為什麼把我一個人丟在這裡！」

賽拉斯（tearful）：「陛下他……每一天都在努力，想讓這一天不要來臨。」

賽拉斯（tearful）：「但他失敗了。臣也失敗了。」

【場景】淚水從賽拉斯的眼眶滑落，滴在染血的盔甲上。他的嘴唇在發抖，但握劍的手反而不再顫抖了——像是做了最後的決定。

賽拉斯（tearful）：「請原諒臣的無能，殿下。」

賽拉斯（tearful）：「如果有來世……臣希望能在一個和平的世界裡，繼續守護您。」

## 2.3 刺殺

```
時間線檔案：prologue/prologue_death.dtl
```

【靜默】**Cut to Silence** — 所有聲音瞬間消失。BGM、火焰聲、遠處的喊殺聲，全部歸零。世界彷彿被按下了靜音鍵。

【場景】畫面慢動作。賽拉斯舉劍。火光在劍刃上流動。

艾蓮娜（sad）：「——」

【內心】我想大喊，但喉嚨發不出聲音。我看著他的眼睛——那雙永遠嚴肅、永遠可靠的眼睛裡，現在只剩下碎裂的光。

【SFX】……0.8 秒的絕對靜默。

【SFX】`sword_stab.ogg` — 極低音量、極短。不是影視劇裡誇張的刺入聲，而是一聲悶響。像是什麼東西被安靜地折斷了。

【場景】畫面開始褪色。從邊緣開始，色彩像是被抽乾，只剩下劍刃上的那一抹紅。

【內心】……不痛。

【內心】只是很冷。

【內心】……為什麼。

【內心】為什麼所有人都拋下了我？

【內心】我到底做錯了什麼……

【轉場】`death_fade` — 紅色從畫面中央向外擴散，吞噬一切，最後沉入完全的黑暗。

## 2.4 死亡情報畫面

【場景】純黑畫面。幾秒鐘的完全黑暗與靜默。

【系統】畫面中央緩緩浮現白色文字：

> **「無知，是最溫柔的死刑。」**

【情報獲得】`intel_city_fall`
> **城破的結局**
> 「王城在一夜之間陷落。敵軍從城外攻入，叛軍與暴民從城內響應。魔法防禦陣早在三天前就已失效。國王戰死於城門。」
