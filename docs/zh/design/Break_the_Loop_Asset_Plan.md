### 5.2 音樂設計原則

- **音樂盒旋律**是全曲核心 motif：每次輪迴醒來重複，但隨輪迴次數增加可微調（加入不協和音、放慢速度、加入回聲）
- **宰相主題**：溫柔的木管旋律（前期），扭曲後變成詭異的變調（撕下偽裝時）
- **真結局**應回收序章史詩 BGM 的旋律片段，但以大調重新編曲——象徵迴圈被打破

## 八、UX 設計要點

### 8.1 核心 UX 流程

```
主選單 → [新遊戲]
  → 序章（純 VN，不可控制）
    → 死亡畫面（獲得情報 × 2）
      → 輪迴轉場（白光）
        → Loop 1 醒來（VN → 可能進入探索模式）
          → 調查/對話 → 壞結局 A
            → 死亡畫面（獲得情報）
              → 輪迴轉場
                → Loop 2 ...
                  → ... → 最終輪迴 → 真結局 → Credits
```

### 8.2 死亡 → 重生循環 UX

| 階段 | 畫面 | 時長 | 互動 |
|---|---|---|---|
| 1. 死亡瞬間 | 紅色褪入黑暗（death_fade shader） | 1-2 秒 | 不可操作 |
| 2. 黑幕 + 公主獨白 | 黑底 + 白字漸顯 | 3-5 秒 | 點擊推進文字 |
| 3. 情報獲得 | 卡片翻轉動畫 + 情報名稱/描述 | 每張卡 2 秒 | 自動 / 點擊加速 |
| 4. 過渡文字 | 「帶著這份記憶……」 | 2 秒 | 不可操作 |
| 5. 輪迴重啟轉場 | 白光閃爍 → 漸入新場景 | 1.5 秒 | 不可操作 |
| 6. 新輪迴醒來 | 公主寢房 + 鳥鳴 + 音樂盒 | — | 恢復互動 |

### 8.3 選擇肢 UX

- **可選項**：白色文字，懸停時邊框發光 + 微放大
- **鎖定項**：灰色文字 + 鎖頭圖示。點擊時顯示提示：「缺少某些情報……」
- **已選過的選項**：💎 可加微小勾選標記，提示玩家上次選過
- **時間限制選擇**：💎 某些關鍵選擇加入倒數條（增加緊張感）

### 8.4 情報日誌 UX

- 按 J 開啟，ESC/J 關閉
- 分四個頁籤（圖示 + 文字）：👤 人物 ｜ 🗡️ 陰謀 ｜ 🔍 線索 ｜ ⚖️ 真相
- 新情報有 **脈衝光暈** 動畫 + 「NEW」標記
- 每條情報：標題 + 來源迴圈標記 + 簡短描述
- 展開後可看詳細內容 + 與其他情報的關聯提示
- 日誌圖示在 HUD 上，有新情報時抖動提醒

### 8.5 輪迴指示器 UX

- 畫面右上角常駐
- 顯示：迴圈符號 ∞ + 當前輪迴數
- 每次死亡重生時，數字 +1 並播放微動畫
- 💎 可加入「Day X / 180」倒計天數

### 8.6 探索模式 UX（如適用）

- 上下左右 / WASD 移動
- 靠近 NPC 時顯示 E/F 互動提示
- NPC 頭上可加感嘆號（有新對話）或問號（調查對象）
- 場景出口用微光/箭頭標示

### 8.7 首次遊玩引導

- 序章自動播放，不需教學
- Loop 1 醒來後顯示簡易提示：
  - 「按 J 查看情報日誌」
  - 「靠近人物按 E 互動」（如有探索模式）
  - 「點擊或按 Enter 推進對話」
- 提示只出現一次，透明度漸變消失

### 8.8 無障礙考量 💎

- 文字速度可調（設定中）
- 字體大小可調
- 自動播放模式（Auto）
- 快轉已讀文字（Skip）
- 對話紀錄回看（Log/History）
