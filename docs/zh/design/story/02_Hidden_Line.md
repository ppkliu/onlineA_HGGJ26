# Break the Loop — 暗線（Hidden Line）

> **本文件定義玩家「主動調查才能發現」的所有隱藏劇情。**
> 暗線的好壞 = 劇情的細節是否足夠細膩。
> 設計暗線的目的是讓玩家感覺「這是我做出的選擇造成的結果」，而不是「這是劇情需要的結果」。

---

## 第一章：資訊溫室設計

> 公主的日常被三層資訊繭房包裹。核心設計目的：**讓玩家感受到「無知不是公主的錯，而是整個體制的產物」。**

```
┌─────────────────────────────────────────────────────┐
│  第一層：溫柔的麻醉劑                                │
│  （宰相與宮廷刻意製造的享樂泡泡）                     │
│  ┌─────────────────────────────────────────────┐     │
│  │  第二層：盲目的安全感                        │     │
│  │  （周遭好人的善意無知）                       │     │
│  │  ┌─────────────────────────────────────┐     │     │
│  │  │  第三層：微小的違和感                │     │     │
│  │  │  （被刻意壓下的真相碎片）            │     │     │
│  │  │  ★ 只有主動追問才能觸碰到 ★         │     │     │
│  │  └─────────────────────────────────────┘     │     │
│  └─────────────────────────────────────────────┘     │
└─────────────────────────────────────────────────────┘
                    ↑ 公主在這裡
```

### 1.1 第一層：溫柔的麻醉劑

> 消耗玩家的行動回合。選擇這些選項 = 浪費時間，零情報。

#### 事件 A：宰相的寶石

**觸發：** 每次輪迴 Day 1 自動觸發。

莉娜（default）：「殿下早安！宰相大人一早就派人送來了南國的寶石呢，說是要給您鑲在新皇冠上。還有，今天下午有莊園的茶會——」

【選擇肢】
- A.「好，讓我看看那些寶石。」→ **【陷阱】** 消耗半天行動，無情報。
- B.「今天的早餐怎麼跟以前不一樣？」→ 前往事件 D。
- C.「莉娜，妳的眼睛怎麼紅紅的？」→ 前往事件 E。
- D.「我想去見父王。」→ 前往事件 C。

#### 事件 B：禮儀導師的說教

**觸發：** 城堡走廊隨機觸發。

禮儀導師（嚴肅）：「殿下，容臣直言——那些粗鄙的戰事，不是公主該關心的。與其在走廊上遊蕩，不如去練習您的刺繡。」

【選擇肢】
- A.「你說的對，我去練刺繡。」→ **【陷阱】** 消耗半天行動。
- B.「閉嘴。」→ 導師離開。獲得自由行動，但「宰相來訪」提前觸發。
- C.「邊境的戰事，具體是什麼狀況？」→ 只得到官方說法，無有效情報。

### 1.2 第二層：盲目的安全感

> 用「善意的無知」阻擋調查。讓玩家發現身邊的好人也幫不上忙。

#### 事件 C：試圖見國王

**觸發：** 選擇「去見父王」。

衛兵甲（恭敬）：「殿下，陛下正在進行緊急的軍務會議，下令任何人不得打擾。」

【選擇肢】
- A.「好吧……」→ 返回。無情報。
- B.「（硬闖）」→ 宰相「恰好」出現攔截。前往事件 C-2。
- C. [需要 `intel_retainer_duty`]「幫我轉告賽拉斯，就說……我知道父王的密令。」→ 賽拉斯當晚秘密來見公主。**解鎖中期關鍵對話。**

#### 事件 C-2：宰相的攔截

莫里昂（concerned）：「殿下放心。陛下日理萬機，但一切都在掌控之中。邊境的那些蠻族不過是蚍蜉撼樹。」

【選擇肢】
- A.「……好吧，謝謝宰相。」→ 無情報。
- B.「宰相大人，您知道魔法防禦陣的事嗎？」→ 莫里昂回答「一切正常，臣每個月都會親自督察。」（**伏筆：** 這正是他破壞魔法陣的機會。獲得 `intel_magic_sabotage` 後重聽此段會恍然大悟。）
- C. [需要 `intel_chancellor_wealth`]「宰相大人，聽說您最近新買了一座莊園？」→ 莫里昂表情一瞬間僵硬。觸發宰相警覺旗標。

#### 事件 F：閨蜜的安撫

**觸發：** 庭園場景中與克蕾雅對話。

克蕾雅：「我爸說了，我們的『絕對防禦魔法陣』可是神明賜予的至寶，連一隻蒼蠅都飛不進來。幾百年來從來沒有失效過！」

克蕾雅：「對了對了，妳聽說宰相大人新買的那座南邊莊園了嗎？聽說花園裡有溫泉呢！」

【選擇肢】
- A.「嗯，妳說的對，我想太多了。」→ 無情報。
- B.「魔法陣幾百年沒失效……但如果有人故意破壞呢？」→ 「妳小說看太多了啦」。無直接情報，但記錄疑問旗標。
- C.「宰相的新莊園？國家不是在打仗嗎，他哪來的錢？」→ **【情報獲得】** `intel_chancellor_wealth`

### 1.3 第三層：微小的違和感

> 遊戲中最關鍵的線索入口。需要玩家主動追問才能觸碰到。

#### 事件 D：餐桌上的異變

**觸發：** 選擇關注早餐。

艾蓮娜：「莉娜，今天的早餐怎麼是醃肉？我記得以前不都是新鮮的烤肉嗎？」

莉娜（forced_smile）：「啊……最近天氣不好，獵戶們沒有收穫。」

【選擇肢】
- A.「好吧。」→ 結束。無情報。
- B.「可是現在是春天，正是狩獵的好季節啊。」→ 繼續追問。

（選 B 延續）→ 兩條路徑：
- [需要 `intel_city_fall`]「莉娜。我需要妳告訴我真話。」→ 莉娜崩潰哭泣，提到糧價暴漲。→ **【情報獲得】** `intel_food_change`
- 「我自己去廚房看看。」→ 發現倉庫空了一大半。→ [需要 `intel_city_fall`] 追問「是不是在把糧食轉為戰備儲糧？」→ **【情報獲得】** `intel_food_change`

#### 事件 E：侍女的眼淚

**觸發：** 選擇關注莉娜的異狀。

艾蓮娜：「莉娜，妳的眼睛怎麼紅紅的？哭過了嗎？」

莉娜（forced_smile）：「才沒有呢！是昨晚看書看太晚了——」

【選擇肢】
- A.「那你要注意休息。」→ 結束。無情報。
- B.「莉娜，妳跟了我五年了。妳在說謊的時候，笑容會往左邊歪。」→ 繼續追問。

（選 B 延續）→ 莉娜崩潰：「我哥哥……被徵去當兵了。整個下城區的年輕男人都被徵走了。戰爭稅也加了三倍……」→ **【情報獲得】** `intel_maid_tears`

#### 事件 G：騎士長的疲憊

**觸發：** 城堡走廊遭遇賽拉斯。

【場景】賽拉斯靠在窗邊擦劍。盔甲上有好幾處未修補的凹痕，眼底有深重的黑眼圈。

【選擇肢】
- A.「好。」→ 結束。無情報。
- B.「你的盔甲怎麼凹了這麼多？」→ 賽拉斯搪塞「訓練新兵時不小心」。無情報，但觸發疑問旗標。
- C. [需要 `intel_assassination`]「賽拉斯……你的劍上，是血嗎？」→ 賽拉斯反應異常，急忙轉身離開。無直接情報，但解鎖最終輪迴關鍵對話選項。

## 第二章：進階調查事件

### 事件 H：質問侍女

**觸發：** 擁有 `intel_maid_tears`，在寢宮中選擇深入追問。

艾蓮娜：「妳哥哥被徵兵的事。不只是妳家對不對？整個下城區都是這樣。」

莉娜（crying）：「……是。但我聽說這都是因為國王陛下要打仗……可是如果不是為了享樂，那那麼多錢去了哪裡？」

**【情報獲得】** `intel_war_conscription`

### 事件 I：跟蹤騎士長

**觸發：** 擁有 `intel_retainer_duty`，在城堡走廊選擇「跟蹤賽拉斯」。

【場景】廢棄倉庫中，三十個年輕人正在接受訓練。他們穿的不是正規軍服。

賽拉斯（default）：「殿下。您不應該在這裡。」

【選擇肢】
- A.「我不走。告訴我真相。」→ 被護送回寢宮。累積信任值。
- B. [需要 `intel_assassination`]「我知道你接到了什麼命令。如果你不想執行那道命令，就幫我。」→ 特殊反應。

（選 B 延續）→ 賽拉斯透露：「陛下有令，不得讓殿下知道任何關於戰事的細節。陛下寧可您恨他，也不願您陷入危險。臣也是。」（無直接情報，但累滿信任值，為最終輪迴收服做準備。）

### 事件 J：下城區初探

**觸發：** 擁有 `intel_maid_tears` 或 `intel_food_change`，選擇前往下城區。

【場景】公主偽裝成平民。下城區街角，布魯諾在演講煽動。

（公主旁觀布魯諾的演講。此事件不觸發情報——公主只是旁觀。情報需要在壞結局 B 中以更慘烈的方式獲得。）

### 事件 K：尋找盲眼老工匠

**觸發：** 擁有 `intel_mob_manipulation` + `intel_war_conscription`

**第一步：打聽**

老婦人：「那個老頭子叫達爾。你去東區的廢井旁邊。但我警告你——不要帶任何人去。他那雙眼睛就是因為知道了不該知道的事才被弄瞎的。」

**第二步：與達爾對話**

達爾（wary）：「你的腳步聲很輕。太輕了。穿的是軟底鞋——不是下城區的人。走吧。」

【選擇肢】
- A.「我是一個想知道真相的人。」→ 不信任，線索停滯。
- B. [需要 `intel_food_change`]「國王把糧食轉成了戰備儲糧。你知道他在準備什麼嗎？」→ 部分信任。
- C. [需要 `intel_food_change` + `intel_mob_manipulation`]「我知道國王在備戰，也知道有人在煽動暴民。你被弄瞎的原因——是因為你幫國王建了什麼東西，對不對？」→ **突破。**

（選 C 延續：達爾的完整證詞）

達爾：「地下防線。兩年前，陛下找到我。他說他知道叛徒存在。所以他要我在城下修一套獨立的防禦系統。」

達爾：「徵兵、加稅、儲糧——全是為了這個。他用自己的名譽當代價，好掩護這項工程。」

達爾：「但他沒料到兩件事。第一——工程完成的第二天，就有人來弄瞎了我的眼睛。那個命令的人說話很輕、很慢，像是在唸什麼好聽的詩。」

達爾：「第二——地下防線的啟動需要三個人同時操作三個不同的機關。如果國王戰死了，就只剩兩個人知道位置。而我已經看不見路了。」

**【情報獲得】** `intel_king_truth` + `intel_magic_sabotage`

### 事件 L：質問宰相

**觸發：** 擁有 `intel_king_truth` + `intel_magic_sabotage`

艾蓮娜：「魔法陣的能源核心，最後一次檢修是什麼時候？」

莫里昂（concerned）：「上個月。臣親自確認過，一切正常。」

艾蓮娜：「兩年前，有一位幫王宮做工的老工匠，被人弄瞎了雙眼。您知道這件事嗎？」

莫里昂（default）：「殿下的消息來源倒是很廣呢。」

莫里昂（default）：「殿下。一位公主，如果在宮廷中說出毫無根據的指控，後果會非常嚴重。甚至可能比上次的『軟禁』更嚴重。」

（此事件後，若未在 3 天內採取進一步行動，觸發壞結局 C。）

## 第四章：可探索物品與環境敘事

> 不影響主線、不觸發情報，但增加沉浸感的細節。讓世界感覺是「活的」。

### 公主寢宮

| 物品 | 描述 | 隱藏含義 |
|---|---|---|
| 書架上被翻爛的童話書 | 《勇敢的小公主》，書角被折了很多次 | 父王以前每晚唸給她聽的，書頁間夾著一片乾燥的花瓣 |

| 物品 | 描述 | 隱藏含義 |
