# Break the Loop — 濾鏡反轉框架（Reversal Framework）

> 本文件定義了 Break the Loop 的核心敘事引擎：**四層濾鏡反轉**。
> 每次輪迴的壞結局不是因為公主犯蠢，而是因為她帶著**合理但不完整的認知**做了看似正確的決策。
> 每一層濾鏡的剝落，都讓公主離真相更近——同時也離崩潰更近。

---

## 框架總覽

```
公主的認知演進：

序章 ──── 「世界毀滅了，所有人背叛了我」
  │          ↓ 濾鏡 1：保護者 = 行刑者
  │
Loop 1 ─── 「父王是暴君，宰相是好人」
  │          ↓ 濾鏡 2：慈父 = 暴君
  │          ↓ 打臉：宰相撕下偽裝開城門
  │
Loop 2 ─── 「宰相是叛徒，平民是可以拯救的」
  │          ↓ 濾鏡 3：受害者 = 暴民
  │          ↓ 打臉：暴民把公主賣給敵軍
  │
Loop 3 ─── 「所有人都不可信，父王是冷血的實用主義者」
  │          ↓ 濾鏡 4：冷酷帝王 = 溫柔獻祭者
  │          ↓ 打臉：弄瞎達爾的是宰相，鎖糧倉的也是宰相
  │
Final ──── 「父王......對不起。換我來保護你了。」
            ↓ 所有濾鏡剝落 → 100% 真相 → 真結局
```

---

## 設計原則

1. **每次死亡都是「合理的錯誤」**：公主在每個迴圈裡做的選擇，從她當時掌握的資訊來看都是合理的。玩家不應該覺得「我被坑了」，而是「原來如此」。

2. **每層濾鏡只揭露部分真相**：壞結局不只是懲罰——它是「學費」。每次死亡都會獎勵一個關鍵情報，讓下一次輪迴能看到更深的層次。

3. **公主的偏見是遞進的**：她不是從頭到尾都在犯同一種錯。每個迴圈她的錯誤類型完全不同——從天真到憤怒到失望到冷酷，最後才到達理解。

4. **最終反轉必須讓所有前置反轉都「說得通」**：真相揭露後，回頭看每一個壞結局，所有的線索和矛盾都應該能被重新解讀。

## 反轉 1：保護者變行刑者

**迴圈：** 序章（Loop 0）
**對應資料夾：** `loop_0_prologue/`

| 項目 | 內容 |
|---|---|
| 公主的初始認知 | 城堡很安全，賽拉斯會保護我，父王會解決一切 |
| 遭遇的部分真相 | 城破了，父王死了，賽拉斯拿劍刺穿了我的心臟 |
| 公主的錯誤結論 | 「世界毀滅了，連最忠誠的騎士都背叛了我，父王無能為力」 |
| 情報獎勵 | `intel_city_fall`（城破事實）、`intel_assassination`（騎士長的刀） |
| 真相 | 賽拉斯不是背叛，而是在執行國王的密令保護她免受更糟的命運 |

**設計要點：** 這是唯一一次「沒有選擇」的死亡。目的是建立最原始的恐懼和「被拋棄」的情緒基底，讓後續所有迴圈的選擇都帶著這份創傷。

## 反轉 2：慈父變暴君，好人變叛徒

**迴圈：** Loop 1
**對應資料夾：** `loop_1/`

| 項目 | 內容 |
|---|---|
| 公主的進場心態 | 「賽拉斯為什麼要殺我？父王在哪裡？我必須找宰相幫忙......」 |
| 宰相的操弄 | 主動接近公主，表現出「唯一願意聽她說話的大人」的姿態。引導她去調查「國王的暴政」 |
| 遭遇的部分真相 | 廚房的鮮肉不見了；莉娜的哥哥被強徵；宰相拿出「國王瘋狂斂財、主動挑釁敵國」的偽證。下城區的人在餓肚子 |
| 公主的錯誤結論 | 「父王是貪婪的暴君！是他引發了戰爭和饑荒！宰相才是試圖拯救國家的人」 |
| 打臉時刻（Bad End A） | 公主幫宰相偷取兵符/癱瘓城防調度 → 宰相立刻撕下偽裝，大開城門引敵軍入城 |
| 死亡後的新認知 | 「宰相是真正的叛徒！父王或許做錯了什麼，但宰相更壞！」 |
| 情報獎勵 | `intel_chancellor_betrayal`（宰相的背叛——親眼看到他開城門）|

**設計要點：**
- 宰相的偽證必須「看起來很真」。他展示的每一份文件都是經過精心偽造的，而且糧食短缺、徵兵等事實確實存在——只是原因被扭曲了。
- 公主幫宰相做的事不能太離譜，必須是「如果國王真的是暴君，這麼做就是正確的」的行動。例如：把兵符交給「忠良的宰相」以便「撥亂反正」。
- Bad End A 的震撼在於：公主以為自己在拯救國家，結果親手加速了它的毀滅。

## 反轉 3：可憐的受害者變嗜血的刁民

**迴圈：** Loop 2
**對應資料夾：** `loop_2/`

| 項目 | 內容 |
|---|---|
| 公主的進場心態 | 「宰相是叛徒！父王瘋了！我只能去靠那些受苦的平民了......」 |
| 公主的行動 | 繞過所有貴族，直接去下城區拯救平民。向布魯諾坦白宰相的陰謀，承諾打開王家糧倉 |
| 遭遇的部分真相 | 平民的痛苦是真實的——飢餓、病痛、失去親人。但他們的憤怒已經失控 |
| 公主的錯誤結論 | 「只要我展現誠意，受苦的平民就會和我站在一起對抗內鬼」 |
| 打臉時刻（Bad End B） | 暴民根本不聽解釋。飢餓與仇恨已經讓他們失去理智，直接把公主綁起來當成向敵軍換取糧食的籌碼 |
| 死亡後的新認知 | 「這些人根本不可理喻......難怪父王要用鐵腕統治，對他們仁慈根本是錯的！」 |
| 情報獎勵 | `intel_mob_manipulation`（暴民被人操弄的證據） |

**設計要點：**
- 公主在這個迴圈對「平民」的認知從同情轉為恐懼和憤怒。這個階段的偏見是最危險的——它會讓公主走向「冷血統治者」的心態。
- 布魯諾的台詞必須讓玩家理解：他不是壞人，他是被逼到絕路的人。但他的行為客觀上毀了公主。
- 敵國士兵收到公主時的反應必須讓公主（和玩家）看到：暴民被承諾的「糧食」根本不存在。

## 反轉 4：冷酷帝王變溫柔的獻祭者

**迴圈：** Loop 3 / 中期輪迴
**對應資料夾：** `loop_3_mid/`

| 項目 | 內容 |
|---|---|
| 公主的進場心態 | 「平民不講理，貴族不可信。沒人可以信任。父王到底是個怎樣的怪物？我必須自己去查清楚」 |
| 公主的行動 | 對所有人都心寒，決定獨自挖出父王真正的秘密。找到達爾，發現地下防線和戰備儲糧 |
| 遭遇的最後誤導 | 儲糧的倉庫鎖著（真相：宰相利用職權扣押了國王要發放的救濟糧）。達爾的眼睛瞎了（公主誤以為是父王為了保密而弄瞎達爾） |
| 公主的錯誤結論 | 「父王沒有瘋，他是個絕對冷血的實用主義者。他犧牲了達爾，餓死了平民，只為了保住王室的火種」 |
| 打臉時刻（Bad End C） | 公主帶著「父王是冷血實用主義者」的理解去行動——只保全內城，放棄下城區。宰相趁虛而入，反撲將公主幽禁 |
| 死亡後的新認知 | 碎片化的線索暗示真相比她想的更複雜。達爾說的那個「說話像在唸詩」的人......不是父王 |
| 情報獎勵 | `intel_king_truth`（國王犧牲的部分真相）、`intel_chancellor_treason`（宰相叛國的間接證據） |

**設計要點：**
- 這是最後一層「錯誤濾鏡」。公主已經非常接近真相了，但最後一步走歪了——因為她在前三個迴圈累積的創傷讓她無法相信「父王是好人」。
- 達爾不會直接說出「是宰相弄瞎我的」——他只描述兇手的特徵。公主因為偏見，把這些特徵往父王身上套。
- Bad End C 的痛苦不在於死亡本身，而在於「我明明已經這麼接近了」。

## 真結局觸發：所有濾鏡剝落

**迴圈：** 最終輪迴
**對應資料夾：** `loop_final/`

| 項目 | 內容 |
|---|---|
| 公主的進場心態 | 帶著四次死亡的記憶和所有碎片化的情報，她終於放下偏見，重新審視所有線索 |
| 100% 真相的拼圖 | 透過達爾的最後一句話 + 賽拉斯的坦白，公主拼出完整真相 |

**100% 真相：**

1. **弄瞎達爾的不是國王，是宰相。** 達爾描述的「說話很輕、很慢，像在唸詩」的人，是宰相的口癖。
2. **鎖住糧倉、讓平民挨餓的不是國王，是宰相利用職權扣押了國王要發放的救濟糧。** 國王每月撥出糧食救濟下城區，但全部被宰相以「戰備需要」的名義截留。
3. **國王假裝暴君、刻意疏遠公主，是因為他知道朝中有內鬼。** 如果他表現得在乎平民和女兒，宰相就會拿他們當籌碼威脅他。
4. **國王背負了所有的罵名、女兒的誤解、平民的仇恨，只是為了把所有資源榨出來，建構能保護「所有人（包含下城區）」的地下防線。**

**公主的最終內心獨白：**

> 「父王......對不起。你一直在孤軍奮戰。這次，換我來保護你了。」

## 與 Canon Line 的對照

每個反轉的「部分真相」都必須符合 Canon Line 的設定，只是**被刻意省略了關鍵環節**：

| 反轉中的部分真相 | Canon Line 的完整真相 | 被省略的環節 |
|---|---|---|
| 「糧食不見了」 | 國王撥糧救濟下城區，被宰相截留 | 宰相截留這一步 |
| 「莉娜哥哥被徵兵」 | 國王徵兵備戰是為了建地下防線 | 備戰的真正目的 |
| 「達爾被弄瞎了」 | 宰相為滅口而弄瞎達爾 | 兇手的身份 |
| 「糧倉被鎖住」 | 宰相以戰備名義截留救濟糧 | 是誰下的鎖 |
| 「國王像暴君」 | 國王故意裝暴君保護所有人 | 偽裝的動機 |
