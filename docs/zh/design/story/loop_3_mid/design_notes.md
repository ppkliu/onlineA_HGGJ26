# Loop 3 / 中期輪迴 — 設計筆記

## 反轉編號與核心命題

**反轉 4：冷酷帝王變溫柔的獻祭者**

這是最後一層錯誤濾鏡。公主已經非常接近真相了——她知道地下防線、知道宰相可能有問題、知道國王在備戰。但三次死亡累積的創傷讓她無法相信「父王是好人」，於是在最後一步走歪了。

## 公主的進場心態

- 帶著 Loop 1 和 Loop 2 的教訓：「貴族是陷阱，平民是刀刃」
- 徹底孤立：不信任任何人，決定獨自行動
- 核心偏見：「父王是個怎樣的怪物？」
- 醒來內心獨白：「平民根本不講理……沒人可以信任。父王到底是個怎樣的怪物？我必須自己去查清楚。」

## 目標情緒曲線

```
冰冷（醒來）→ 孤獨（獨自行動）→ 震撼（發現地下防線）→ 錯誤理解（糧倉/達爾）→ 冷血決策（放棄下城區）→ 被反撲 → 最後一秒的頓悟 → 不甘
```

情緒最高點：發現地下防線的那一刻——「原來父王一直在準備」
情緒最低點：塔樓崩塌前一秒——「差一步，只差最後一步」

## Bad End 觸發條件

- 主線觸發：公主在 Day 90-120 區間孤身行動，被宰相察覺後反撲 → 幽禁至城破 → Bad End C
- 替代路線：如果公主在質問宰相前先聯繫了賽拉斯（需要足夠的信任值），可以獲得賽拉斯的保護。但因為公主此時不信任任何人，大部分玩家不會選這條路。

## 與舊版差異對照

| 項目 | 舊版 | 新版（反轉框架）|
|---|---|---|
| 公主去找達爾的動機 | 收集情報 | 「所有人都不可信，我要自己查父王」 |
| 對鎖住糧倉的理解 | 知道是戰備儲糧 | 發現財政署印記指向宰相，卻得出「父王也用冷血犧牲換保護，那我也用同樣的邏輯」的錯誤結論 |
| 對達爾失明的理解 | 知道有人害他 | 排除了父王（聲音不對），卻無法把「唸詩的聲音」和宰相連結——拼圖差最後一塊 |
| Bad End 的根因 | 「一個人力量不夠」 | 「知道有內鬼卻走向了冷血主義」+「一個人力量不夠」 |

## 設計備忘

- 糧倉上的「財政署」印記是關鍵伏筆。公主在 Loop 3 會注意到並聯想到「財政署=宰相直管」，但無法做出更進一步的結論，反而用「父王也在冷血犧牲，所以我也用同樣的邏輯」說服自己。在最終輪迴才恍然大悟：截留糧食、弄瞎達爾、腐蝕魔法陣的都是宰相，而非父王的授意。
- 達爾描述的「唸詩般的聲音」是貫穿全劇的核心線索。宰相在 Loop 1 花園中和公主說話時用的就是這種語調——但公主當時正在被操弄，不會把兩者聯繫起來。
- Bad End C 最後一刻的「頓悟」不能太完整。公主只是隱約感覺到「那個聲音像宰相」，但來不及確認就死了。這個「差一步」的遺憾，是推動最終輪迴的最強動力。
- 在 Bad End C 的塔樓中，如果玩家仔細觀察，可以看到窗外遠方有一個小人影在向城堡方向奔跑——那是賽拉斯從北方趕回來，但已經來不及了。
