# Loop 3 / 中期輪迴 — 暗線設計

> Loop 3 的暗線核心：**拼湊國王秘密的碎片**。
> 公主在這個迴圈中獲得最接近真相的情報，但因為偏見做出了錯誤的解讀。

---

## 調查事件

### 事件 K：尋找盲眼老工匠

**觸發：** 擁有 `intel_mob_manipulation`（知道暴民被操弄）+ `intel_war_conscription`（知道徵兵的規模）。

**第一步：打聽**

在下城區詢問「有沒有認識被弄瞎眼的老工匠」。需要避開布魯諾的勢力範圍。

老婦人：「達爾。東區廢井旁邊。但不要帶任何人去。」

**第二步：與達爾對話**

達爾（wary）：「你的腳步聲很輕。太輕了。」

【選擇肢】
- A.「我是一個想知道真相的人。」→ 不信任，只透露少量資訊。
- B. [需要 `intel_food_change`]「國王把糧食轉成了戰備儲糧。你知道他在準備什麼嗎？」→ 部分信任。透露地下防線存在。
- C. [需要 `intel_food_change` + `intel_mob_manipulation`]「我知道國王在備戰，也知道有人在煽動暴民。你被弄瞎的原因——是因為你幫國王建了什麼東西，對不對？」→ **突破。** 完整證詞。

**選 C 延續：達爾的完整證詞**

達爾透露：
1. 國王兩年前委託他建造地下防線
2. 防線需要三個人同時操作三個機關
3. 完工後第二天他就被弄瞎了
4. 兇手的特徵：「說話很輕、很慢，像在唸詩」

**【情報獲得】** `intel_king_truth`（國王犧牲的部分真相）

### 事件 K-2：探索地下防線

**觸發：** 獲得 `intel_king_truth` 後，從達爾處得知入口位置。

公主在地下發現：
1. 完整的機關防禦系統
2. 鎖住的大型糧倉
3. 防線啟動需要三個機關同時操作的標示

**環境線索：**
- 糧倉的鎖上有特殊標記——不是王家紋章，而是「財政署」的印記
- 鎖的製造日期比防線完工晚了三個月

**【情報獲得】** `intel_magic_sabotage`（如果仔細查看地下入口附近的能源管線——有被動過手腳的痕跡）

### 事件 L：質問宰相

**觸發：** 擁有 `intel_king_truth` + `intel_magic_sabotage`

艾蓮娜：「魔法陣的能源核心，最後一次檢修是什麼時候？」

莫里昂（concerned）：「上個月。臣親自確認過，一切正常。」

艾蓮娜：「兩年前，有一位幫王宮做工的老工匠，被人弄瞎了雙眼。您知道這件事嗎？」

莫里昂（default）：「殿下的消息來源倒是很廣呢。」

莫里昂（cold）：「殿下。一位公主，如果在宮廷中說出毫無根據的指控，後果會非常嚴重。甚至可能比上次的『軟禁』更嚴重。」

**（此事件後，若未在 3 天內採取進一步行動，觸發 Bad End C。）**

## 關鍵設計：公主的錯誤解讀

公主在 Loop 3 中收集到的情報本身是正確的，但她的**解讀**被三次死亡累積的偏見扭曲了：

| 正確情報 | 公主的錯誤解讀 | 真相 |
|---|---|---|
| 國王建了地下防線 | 國王是冷血的實用主義者 | 國王是為了保護所有人 |
| 糧倉被鎖住 | 國王放棄下城區，只保內城 | 宰相截留了國王要發的救濟糧 |
| 達爾被弄瞎 | 國王為了保密不擇手段 | 是宰相為了滅口 |
| 達爾描述的「唸詩般的聲音」 | 國王偽裝聲音（合理化偏見） | 那就是宰相的說話方式 |

公主要到 Bad End C 的最後一刻才隱約意識到這個錯誤——但已經太遲了。
