# Loop 2 — 設計筆記

## 反轉編號與核心命題

**反轉 3：可憐的受害者變嗜血的刁民**

Loop 2 的核心衝突：公主帶著對平民的同理心行動，卻發現善意在極端環境下是脆弱的。暴民不是壞人——他們是被逼到牆角的好人。但被逼到牆角的好人，可能比壞人更危險。

## 公主的進場心態

- 帶著 Loop 1 的教訓：「宰相是叛徒，不能信任貴族」
- 但還不知道父王的真實動機
- 核心偏見：「平民是受害者，他們和我站在同一邊」
- 醒來內心獨白：「宰相是叛徒！父王瘋了！我不能信任任何貴族。那些受苦的平民……他們和我一樣是受害者。」

## 目標情緒曲線

```
冷靜（醒來，帶著教訓）→ 決心（去下城區）→ 融入（與平民生活）→ 同理（理解痛苦）→ 信任（坦白身份）→ 背叛（被出賣）→ 心碎
```

情緒最高點：坦白身份後布魯諾說「給我三天」——以為取得了信任
情緒最低點：被綁起來交給敵軍——「我說過了」的無聲眼神

## 設計備忘

- 布魯諾不是反派。他在「保護兄弟們」和「保護公主的信任」之間做了一個痛苦的選擇。他最終選擇了前者——因為他的兒子在挨餓。
- 酒窖裡的「審判」必須讓玩家感受到群體暴力的可怕：一開始只有一個人提出「把公主交出去」，但恐懼和飢餓讓越來越多人附和。
- 敵國士官的「打下城之後再說」是關鍵台詞。它讓布魯諾（和玩家）意識到：暴民被騙了。但已經太遲了。
- 公主在 Bad End B 後的偏見轉向是最危險的——「平民不可理喻，鐵腕才是對的」。這個偏見會讓她在 Loop 3 中做出新的錯誤。
