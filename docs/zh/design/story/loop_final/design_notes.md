# 最終輪迴 — 設計筆記

## 核心命題

**所有濾鏡剝落。**

公主帶著四次死亡的記憶和完整的真相覺醒。她不再被任何偏見束縛——不再天真、不再被操弄、不再天真地信任、不再冷酷地孤立。她成為了一個真正的領導者。

## 公主的進場心態

- 帶著所有迴圈的教訓和所有情報
- 放下了對父王的怨恨：「他一直在孤軍奮戰」
- 理解了孤軍奮戰的代價：「一個人知道真相是不夠的」
- 醒來內心獨白：「父王……對不起。你一直在孤軍奮戰。這次，換我來保護你了。」

## 目標情緒曲線

```
鋼鐵般的決意（醒來）→ 感動（收服賽拉斯）→ 心酸（安撫布魯諾）→ 痛快（揭露宰相）→ 漫長的等待（180天）→ 釋然（破曉）
```

情緒最高點：賽拉斯撕碎密令——「這是第一次，臣不需要用劍來守護您了」
情緒最低點：國王的轉述——「朕不後悔保護你。只後悔低估了你」

## 三步設計的敘事邏輯

最終輪迴的三個步驟對應了公主在前三個迴圈中學到的教訓：

| 步驟 | 對應教訓來源 | 運用方式 |
|---|---|---|
| 收服賽拉斯 | Loop 0 — 賽拉斯是被逼殺她的 | 用理解而非命令贏得忠誠 |
| 安撫暴民 | Loop 2 — 暴民被操弄、善意不夠 | 用真相 + 實際承諾而非空話 |
| 揭露宰相 | Loop 1 — 宰相的操弄手法；Loop 3 — 證據線索 | 一次性出示鐵證，不給反擊機會 |

## 前幾個迴圈濾鏡回收的設計

最終輪迴中有多處「回收」前幾個迴圈的元素，讓玩家感受到濾鏡的完整剝落：

| 回收元素 | 原始迴圈 | 最終輪迴中的新意義 |
|---|---|---|
| 賽拉斯的眼淚 | Loop 0 — 以為是背叛 | 理解他是因為不捨而流淚 |
| 宰相的溫柔語調 | Loop 1 — 以為是關心 | 就是弄瞎達爾時「像唸詩」的聲音 |
| 布魯諾的憤怒 | Loop 2 — 以為是忘恩負義 | 理解是絕望中被操弄的結果 |
| 鎖住的糧倉 | Loop 3 — 以為是父王的冷血 | 是宰相的截留——鎖上印的是財政署的章 |
| 音樂盒的走音 | 全程 — 詭異的 BGM | 最終完整演奏——迴圈被打破 |

## 設計備忘

- 收服賽拉斯的場景是全遊戲最重要的角色互動。它不只是「展示情報」，而是公主第一次以「完全理解一個人的痛苦」的方式去說服別人。
- 安撫布魯諾的關鍵在「歸還」這個詞。公主不是在「施捨」糧食——她是在說「這本來就是你們的」。這個措辭讓布魯諾的自尊被保全。
- 揭露宰相時莫里昂最後的「欣賞」表情不是隨便加的。他是一個精於算計的人，在被打敗的瞬間反而佩服公主——因為他知道要走到這一步需要多少次死亡和多少教訓。
- 真結局的核心情緒不是「勝利」而是「釋然」。BGM 不是凱旋曲——而是安靜的鋼琴。因為代價已經付出了（公主死了很多次），這不是一場痛快的勝利。
- 撕碎密令的碎片像白色蝴蝶飛散——這是從 Loop 0 中「被劍貫穿」的殘酷到「被風帶走」的溫柔之間，最完美的對比。
