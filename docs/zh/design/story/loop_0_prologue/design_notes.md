# Loop 0：序章 — 設計筆記

## 反轉編號與核心命題

**反轉 1：保護者變行刑者**

公主從「城堡很安全，騎士長會保護我」的認知，被暴力地拉進「連最忠誠的人都要殺我」的現實。這是唯一一次沒有選擇的死亡——建立所有後續迴圈的情緒基底。

## 公主的進場心態

- 完全無知。不知道戰爭、不知道魔法陣失效、不知道父王的犧牲。
- 對賽拉斯的信任是絕對的——他是從小保護她的人。
- 對父王的期待是：「他一定會派人來救我。」

## 目標情緒曲線

```
混亂 → 恐懼 → 希望（看到賽拉斯）→ 困惑 → 絕望 → 怨恨 → 死亡
```

情緒最高點：看到賽拉斯進門的那一刻（以為得救了）
情緒最低點：賽拉斯舉劍的那一刻（Cut to Silence）

## 設計備忘

- 序章的功能是「建立創傷」，不是「給資訊」。玩家在這裡不需要理解為什麼城破了，只需要感受到恐懼和被拋棄。
- Cut to Silence 是整個遊戲最重要的音效設計時刻。0.8 秒的絕對靜默必須讓玩家的心臟停跳。
- 賽拉斯的台詞在序章裡聽起來像是背叛，但在最終輪迴回顧時，每一句都是心碎的告白。
