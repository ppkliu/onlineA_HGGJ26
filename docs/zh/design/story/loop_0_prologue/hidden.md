# Loop 0：序章 — 暗線設計

> 序章沒有調查事件——公主在這裡只有死亡和恐懼。
> 暗線在序章的作用是「埋伏筆」：玩家在後續迴圈中回想序章時會發現的細節。

---

## 隱藏在序章中的伏筆

這些細節在第一次玩時不會被注意到，但在後續迴圈中（帶著情報重看）會產生全新的意義：

| 細節 | 第一次看到時的理解 | 帶著情報回看時的理解 |
|---|---|---|
| 「魔法陣在三天前就失效了」 | 不明原因的災難 | 宰相在 Day 177 完成了最後一次破壞 |
| 「有人從內部切斷了能源核心」 | 有內鬼，但不知道是誰 | 宰相利用每月督察的機會注入腐蝕劑 |
| 「南門、西門都被打開了——是城裡的人」 | 平民暴動 | 布魯諾被宰相操弄，帶暴民開了門 |
| 「陛下已經戰死在城門前」 | 國王無能，死了 | 國王明知必敗仍然親自上陣，為了給賽拉斯爭取時間 |
| 賽拉斯的眼淚 | 背叛者的做戲 | 一個被迫殺死自己守護對象的人的崩潰 |
| 「如果有來世」 | 空洞的告別 | 他真心的願望——在一個不需要這道密令的世界裡繼續守護她 |

## 序章中不可見的 NPC 行動（Canon 參考）

在公主被困在寢房的同時，城堡外正在發生：

| 時間 | 事件 |
|---|---|
| 日落 | 敵軍發動總攻。魔法陣無效，城牆防禦僅靠物理結構 |
| 入夜 | 布魯諾帶領暴民打開南門。宰相的人同時打開西門 |
| 深夜 | 國王在城門前率殘軍抵抗。賽拉斯被迫離開國王身邊，奉命去寢房 |
| 凌晨 | 國王戰死。賽拉斯殺穿走廊敵兵到達公主寢房。莫里昂從密道逃離 |

## 可探索物品（序章場景中）

序章是線性劇情，但寢房中有可互動物品（不影響劇情，純環境敘事）：

| 物品 | 描述 | 隱藏含義 |
|---|---|---|
| 窗邊的花瓶 | 插著已經枯萎的花 | 莉娜最後一次來澆水是三天前——城堡已經開始混亂了 |
| 床頭的音樂盒 | 被震動從架上摔落，摔壞了 | 音樂盒的旋律就是輪迴 BGM。壞掉 = 輪迴的不完美 |
| 門上的鎖 | 從外面鎖住的 | 誰鎖的？不是敵軍——是城堡衛兵在執行「保護公主」的常規措施。諷刺的是，這個「保護」讓她無法逃跑 |
