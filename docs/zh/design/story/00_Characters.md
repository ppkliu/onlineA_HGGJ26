# Break the Loop — 角色總表與勢力關係

> 本文件定義所有角色的設定、勢力歸屬、以及 6 個月間的行為軌跡。
> 角色設計的最高原則：**沒有純粹的壞人，只有被處境逼到極端的人。**

---

## 核心角色

### 公主 艾蓮娜（Princess Elena）

| 項目 | 說明 |
|---|---|
| 年齡 | 17 歲 |
| 勢力歸屬 | 王室（但實質上不屬於任何勢力，是所有勢力的棋子） |
| 角色定位 | 玩家角色。從無知的籠中鳥，到看透一切的破局者 |
| 口癖/語氣 | 初期：天真、疑問句多。後期：冷靜、陳述句，偶爾帶著輪迴疲憊的蒼涼 |
| 表情組 | `default`（微笑）、`sad`（哀傷）、`shocked`（驚愕）、`angry`（憤怒）、`determined`（堅定）、`hollow`（空洞，多次輪迴後） |
| 核心衝突 | 她是這座城堡裡唯一「什麼都不知道」的人——而這份無知，是所有人刻意維護的 |

**角色弧線：**

| 階段 | 心態 | 行動模式 |
|---|---|---|
| 序章（Loop 0） | 等待被拯救的公主 | 被動等待 → 死於無知。**濾鏡 1：保護者=行刑者** |
| Loop 1 | 抓住稻草、被宰相操弄 | 信任宰相 → 交出兵符 → 親手毀了國家。**濾鏡 2：慈父=暴君** |
| Loop 2 | 繞過貴族、同情平民 | 直接救平民 → 被暴民出賣。**濾鏡 3：受害者=暴民** |
| Loop 3 | 不信任任何人、獨自行動 | 找到達爾但誤解父王。**濾鏡 4：冷酷帝王=獻祭者** |
| 最終輪迴 | 所有濾鏡剝落，不再等人來救 | 收服盟友 → 安撫暴民 → 揭露宰相 → 主動拯救所有人 |

### 騎士長 賽拉斯（Knight Commander Silas）

| 項目 | 說明 |
|---|---|
| 年齡 | 34 歲 |
| 勢力歸屬 | 友善貴族（王室忠臣） |
| 角色定位 | 序章的「加害者」，真相中的「最大受害者」。整個遊戲最令人心碎的角色 |
| 口癖/語氣 | 寡言、用軍事化的簡短句。情緒崩潰時才會說長句 |
| 表情組 | `default`（嚴肅）、`tearful`（含淚）、`exhausted`（疲憊）、`relieved`（釋然） |
| 核心衝突 | 從小看著艾蓮娜長大。國王的密令要求他在城破時殺死公主。他每天都在祈禱不需要執行這道命令 |

**關鍵設計：** 賽拉斯在日常中的每一句冷淡回應，都是因為他不敢和公主太親近——因為他知道自己可能要親手殺她。玩家在後期輪迴理解這一點後，重聽他前期的台詞會極度心痛。

**6 個月行為軌跡：**

| 月份 | 明線可見行為 | 真實行為 |
|---|---|---|
| Month 1 | 冷淡、公事公辦地打發公主 | 夜間秘密訓練第 2 批民兵（約 30 人） |
| Month 2 | 偶爾在走廊出現，盔甲有凹痕 | 城外伏擊敵軍斥候，殺了 2 名敵軍探子 |
| Month 3 | 看起來極度疲憊，拒絕所有追問 | 民兵擴編至 50 人，與斥候小隊交手負傷 |
| Month 4 | 被宰相調離王城（若未被收服） | 被迫北上，但秘密留下副手繼續訓練 |
| Month 5 | 不在城中（若未被收服） | 在北方邊境得知城內異變，拼命趕回 |
| Month 6 | 含淚出現在公主寢房 | 執行最後密令——或被公主收服後並肩作戰 |

### 宰相 莫里昂（Chancellor Morion）

| 項目 | 說明 |
|---|---|
| 年齡 | 52 歲 |
| 勢力歸屬 | 敵意貴族（叛國者） |
| 角色定位 | 真正的幕後黑手。玩家前期的「假盟友」 |
| 口癖/語氣 | 永遠溫和有禮，用「殿下」開頭。越危險的話說得越輕柔 |
| 表情組 | `default`（慈祥微笑）、`concerned`（假裝擔憂）、`cold`（露出真面目，僅最終輪迴） |
| 核心衝突 | 國王的改革削減了貴族特權。與敵國密約：裡應外合攻破王城，事後保留封地與財富 |

**關鍵設計：** 莫里昂的台詞永遠不說謊——他只是省略真相。例如「陛下已經派人處理了」（真相：國王確實在備戰，但莫里昂在暗中破壞）。玩家在獲得情報後重讀他的話，會發現每一句都是精心設計的文字陷阱。

**6 個月行為軌跡：**

| 月份 | 明線可見行為 | 真實行為 |
|---|---|---|
| Month 1 | 送寶石、安排茶會、關心公主 | 第 3 次對魔法陣能源核心注入腐蝕劑；向敵國傳遞第 1 封密函 |
| Month 2 | 攔截公主見國王，安撫「一切正常」 | 收到敵國回信，敲定合作框架；佈置對公主的監視 |
| Month 3 | 透過禮儀導師加強對公主的行動限制 | 派人弄瞎達爾；代理人接觸布魯諾 |
| Month 4 | 提議「加強公主安保」；調離賽拉斯 | 與敵國確定裡應外合日期；排擠友善貴族 |
| Month 5 | 在朝會上表現得「憂心忡忡」 | 完成魔法陣最後破壞；安排逃跑路線 |
| Month 6 | 消失在混亂中 | 帶著財產從密道逃離（若未被逮捕） |

### 侍女 莉娜（Maid Lina）

| 項目 | 說明 |
|---|---|
| 年齡 | 19 歲 |
| 勢力歸屬 | 無知平民（受害者） |
| 角色定位 | 公主身邊最近的人，也是「資訊溫室」的第一道牆 |
| 口癖/語氣 | 活潑、愛八卦。但偶爾會突然沉默，然後用更大的笑容掩飾 |
| 表情組 | `default`（開朗）、`forced_smile`（勉強微笑）、`crying`（哭泣）、`grateful`（感激） |
| 核心衝突 | 哥哥被強徵入伍，家裡因戰爭稅吃不起飯。但她不能在公主面前表露，因為「讓殿下開心」是她唯一能做的事 |

**關鍵設計：** 莉娜不是內鬼，她是體制的受害者。她用笑容包裝痛苦，和公主用無知包裝恐懼，是同一種悲劇的兩面。

**6 個月行為軌跡：**

| 月份 | 明線可見行為 | 真實行為 |
|---|---|---|
| Month 1 | 開朗如常，但眼睛偶爾紅腫 | 偷偷把省下的薪水寄回家 |
| Month 2 | 開始偶爾遲到，說「家裡有點忙」 | 母親病了，哥哥被正式編入前線部隊 |
| Month 3 | 有一天沒有來送餐 | 得知哥哥在邊境受傷的消息 |
| Month 4 | 變得沉默，不再主動聊天 | 被宰相的手下警告「不要跟公主說太多」 |
| Month 5 | 手上出現淤青（若追問會否認） | 因為幫公主準備偽裝被宰相的人發現並威脅 |
| Month 6 | 最後一天沒有出現 | 在城破之夜被困在下城區（若公主未介入）；或哥哥平安歸來（真結局） |

### 盲眼老工匠 達爾（Blind Craftsman Dal）

| 項目 | 說明 |
|---|---|
| 年齡 | 67 歲 |
| 勢力歸屬 | 知曉真相的平民 |
| 角色定位 | 破局的鑰匙。唯一能告訴公主「國王真正在做什麼」的人 |
| 口癖/語氣 | 慢條斯理，用隱喻和比方說話。不直接回答問題，需要公主用正確的情報去「敲門」 |
| 表情組 | `default`（平靜）、`wary`（警戒）、`sad_smile`（悲傷的微笑） |
| 核心衝突 | 幫國王秘密修建地下防禦設施的總工匠。工程完成後被莫里昂派人弄瞎雙眼滅口，僥倖活下來躲在下城區 |

**關鍵設計：** 達爾的眼睛就是被莫里昂弄瞎的——這個事實本身就是宰相叛國的物證。但達爾不會主動說出來，因為他怕連累收留他的窮人們。公主必須帶著足夠的情報，讓他相信「這個公主不一樣了」。

**6 個月行為軌跡：**

| 月份 | 真實行為 |
|---|---|
| Day 1 前 | 完成地下防線驗收，向國王交付啟動方法 |
| Day 45（Month 2） | 被莫里昂的手下弄瞎雙眼，躲入下城區東區廢井旁的廢墟 |
| Month 3-5 | 在廢墟中靠周圍窮人接濟生活，不見任何外人 |
| Month 6 | 若公主找到他：提供地下防線啟動方法。若無人找到：在城破之夜被瓦礫埋沒 |

### 鐵匠 布魯諾（Blacksmith Bruno）

| 項目 | 說明 |
|---|---|
| 年齡 | 28 歲 |
| 勢力歸屬 | 受鼓動暴民 |
| 角色定位 | 城破之夜打開城門的內應。但他不是壞人——他是被絕望和謊言逼到絕路的普通人 |
| 口癖/語氣 | 粗獷、直接、充滿怒氣。罵的最多的詞是「那些貴族」 |
| 表情組 | `default`（憤怒）、`grieving`（悲痛）、`hesitant`（猶豫）、`hopeful`（希望） |
| 核心衝突 | 妻子難產而死（草藥被徵為軍需），兒子餓到生病。宰相的人告訴他「敵國會帶來糧食」 |

**關鍵設計：** 布魯諾是整個遊戲中最難應對的角色。他的憤怒是真實的、合理的。公主不能用「國王是好人」這種空話說服他——她必須帶來國王確實在為平民準備後路的具體證據。

**6 個月行為軌跡：**

| 月份 | 明線可見行為 | 真實行為 |
|---|---|---|
| Month 1 | 不可見（公主尚未走出城堡） | 妻子剛死不久，開始在街頭痛罵王室 |
| Month 2 | 不可見 | 收到宰相代理人的「外部支援」傳單 |
| Month 3 | 在下城區廣場演講煽動 | 開始組織暴民，與宰相的代理人定期聯絡 |
| Month 4 | 演講規模擴大，跟隨者增多 | 取得攻城門的工具，制定行動計劃 |
| Month 5 | 暴民開始武裝巡邏 | 收到宰相傳來的「行動日期」 |
| Month 6 | 打開城門（若未被安撫）| 或：加入民兵志願隊守城（真結局） |

### 國王（不直接登場，但影響一切）

| 項目 | 說明 |
|---|---|
| 年齡 | 48 歲 |
| 角色定位 | 缺席的父親。他的每一個決定都是為了保護城市和女兒，但代價是讓所有人誤解他 |
| 核心衝突 | 「朕不知道叛徒是誰，但朕知道叛徒存在」——所以他選擇犧牲名譽秘密備戰，而非打草驚蛇 |

**國王不直接出現在公主面前**（直到最終輪迴的朝堂對質），但他的影響無處不在：徵兵令是他下的，稅是他加的，密令是他給的，地下防線是他建的。他是整個故事中最大的「善意加害者」。

### 三層衝突結構

| 層次 | 衝突 | 遊戲影響 |
|---|---|---|
| 上層：權力暗戰 | 莫里昂 vs. 賽拉斯 | 公主的政治盟友選擇 |
| 中層：資訊操弄 | 莫里昂的謠言 vs. 達爾的真相 | 公主的情報收集路線 |
| 下層：生存危機 | 布魯諾的絕望 vs. 國王的犧牲 | 公主能否在城破前化解內亂 |
