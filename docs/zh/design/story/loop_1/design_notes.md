# Loop 1 — 設計筆記

## 反轉編號與核心命題

**反轉 2：慈父變暴君，好人變叛徒**

Loop 1 是整個遊戲中改動最大的迴圈。舊版的 Loop 1 是「公主冒進闖入王座廳被軟禁」——這讓公主看起來很蠢。新版中，公主的每一步都是「在宰相的引導下做出合理的判斷」，這才符合濾鏡反轉框架的設計哲學。

## 公主的進場心態

- 帶著序章的創傷：死亡、被拋棄、賽拉斯的劍
- 核心恐懼：「為什麼賽拉斯要殺我？」
- 抓住稻草的心理：宰相是唯一主動對她表達善意的人
- 醒來內心獨白：「賽拉斯為什麼要殺我？父王在哪裡？那個一直對我很溫柔的宰相……他會幫我的。」

## 目標情緒曲線

```
震驚（醒來）→ 不安 → 抓住稻草（宰相）→ 被引導的憤怒（看偽證）→ 心痛（下城區）→ 決意（偷兵符）→ 崩潰（宰相撕下偽裝）→ 自責
```

情緒最高點：偷到兵符的那一刻——「我正在拯救國家」的使命感
情緒最低點：宰相開城門——「是我親手毀了一切」

## Bad End 觸發條件

- 主線觸發：公主在 Day 5 前將兵符交給宰相 → Day 6 觸發 Bad End A
- 替代路線：公主在 Day 3 前累積 `flag_suspicion_ledger` + `intel_retainer_warning` → 可選擇不交出兵符 → 宰相直接軟禁公主 → 回歸舊版壞結局（被軟禁至城破）

## 與舊版差異對照

| 項目 | 舊版 | 新版（反轉框架）|
|---|---|---|
| 公主的行為 | 冒進闖入王座廳大喊「城會陷落」 | 被宰相引導，理性地一步步走向陷阱 |
| 被抓的方式 | 被當瘋子軟禁 | 自願交出兵符，親手造成災難 |
| 宰相的角色 | 被動反應的看門人 | 主動操弄的陰謀家 |
| 公主的感受 | 「沒人聽我說話」（無力） | 「我親手害了所有人」（自責） |
| 情報獎勵 | `intel_retainer_duty` | `intel_chancellor_betrayal`（更強的動力） |

## 設計備忘

- 宰相的偽證必須足夠精巧。帳簿的數字確實有問題（國王的秘密備戰開支不在正式帳簿上），但原因不是「貪污」而是「無法公開的軍事支出」。
- 公主去下城區不是偶然——是宰相安排的。他確保公主看到了最慘的畫面，但看不到任何「國王在暗中救濟」的證據（因為那些救濟都被宰相截留了）。
- Bad End A 的核心教訓：「不要相信主動送到手裡的答案。」
